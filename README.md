# nCode
A simple web-based file editor.

## Requirements
- Global installation of typescript. `npm i -g typescript`
- Yarn

## Setup
1. Frontend
```
cd frontend
yarn
yarn build
```
2. CLI
```
cd cli
yarn
tsc
```

## Run
In the cli folder, run `node build/index.js`.
To install globally, run `npm i -g .` inside the cli folder.
